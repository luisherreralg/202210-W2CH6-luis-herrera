BIENVENIDO

Bienvenido al pasapalabra de ISDI Coders. La partida tiene un límite de tiempo de 130s, una vez terminado el tiempo la partida finalizará. Para jugar solo hay que introducir la palabra que creas correcta y pulsar enter. Si quieres dejar alguna palabra para la siguiente vuelta, solo tienes que escribir "pasaplabra" o "pasa". ¡Mucha suerte!
